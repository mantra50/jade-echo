## 三、规格总览

### 输出格式
- **文件格式：** PNG（透明资源）/ JPG（背景图）
- **色彩空间：** sRGB
- **分辨率：** 棋盘 1080×1920，卡牌 540×780
- **DPI：** 300（用于印刷）/ 72（屏幕用）

### Unity 导入设置
```
Filter: Transparent/Cutout
Max Size: 2048
Compression: High Quality
Generate Mip Maps: ❌（UI 关闭）
```
