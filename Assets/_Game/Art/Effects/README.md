# 特效清单 — 供动画师使用

本文档描述游戏中所有关键特效的动画描述，供动画师实现或外包制作。

---

## 一、消除特效（Match Effect）

### 1.1 普通消除（Gem Clear）

**触发时机：** 3颗以上同色棋子匹配时

**动画描述：**
- **Phase 1（0–150ms）：** 棋子发光，从内向外扩散白色辉光（scale 1.0→1.3，ease-out）
- **Phase 2（150–300ms）：** 棋子膨胀后快速收缩（scale 1.3→0，ease-in）
- **Phase 3（200–500ms）：** 粒子爆发，8–12个小光点向外飞散，轨迹为贝塞尔曲线
- **色彩：** 根据元素属性（火=橙红，水=天蓝，风=浅绿，光=金黄，暗=紫罗兰，土=棕色）
- **音效建议：** 短促清脆的"叮"声，pitch 随机偏移 ±10%

**参数：**
```
Duration: 500ms
Particle Count: 10
Spread Radius: 80px
Easing: ease-out (Phase1), ease-in (Phase2)
```

---

### 1.2 连锁消除（Chain Clear）

**触发时机：** 消除后新棋子下落再次触发匹配（2连锁以上）

**动画描述：**
- 每增加一连锁，消除特效规模逐级放大
- **2连锁：** 基础规模 × 1.2，白色光圈扩散
- **3连锁：** 基础规模 × 1.5，附加震屏（Camera Shake，amplitude 5px，duration 200ms）
- **4连锁+：** 基础规模 × 2.0，附加屏幕闪白（opacity 30%→0，200ms）
- **连锁数显示：** 右上角出现 "×2" "×3" 文字，放大淡出（scale 1.5→0.8，opacity 1→0，duration 800ms）

**参数：**
```
Chain 2: scale 1.2x, white ring expand
Chain 3: scale 1.5x + camera shake (5px, 200ms)
Chain 4+: scale 2.0x + screen flash (30% white, 200ms)
```

---

### 1.3 十字消除（Cross Clear）

**触发时机：** 消除形成十字形（横向+纵向同时满足）

**动画描述：**
- 消除时在交叉点生成一个**更大的光球**
- 光球脉冲动画：scale 0→2→0，duration 600ms
- 交叉的两条光带（横向+纵向）同时向外延伸，fade out
- 附加金色火花粒子从交叉点飞散

## 二、卡牌打出特效（Card Play Effect）

### 2.1 手牌选中高亮

**触发时机：** 玩家点击手牌，将其选中为待打出状态

**动画描述：**
- 卡牌从手牌区域**悬浮上升**（translateY -20px，200ms ease-out）
- 卡牌边框发出**元素对应色**的光芒（glow 0→8px，opacity pulse 0.5↔1.0，loop）
- 手牌区域其他卡牌略微**压暗**（brightness 1.0→0.6，200ms）

---

### 2.2 卡牌打出（Card Launch）

**触发时机：** 玩家确认打出卡牌，卡牌飞向目标位置

**动画描述：**
- 卡牌从手牌区域**斜向上飞出**（bezier 曲线，duration 400ms，ease-out）
- 飞行过程中卡牌轻微**旋转**（rotation 0→15°→0）
- 飞出时拖尾：**元素色残影**（3张半透明卡牌影子，间隔 50ms）
- 落地目标位置时产生**冲击波**（scale 0.5→2.0，opacity 0.8→0，ring shape，300ms）

**参数：**
```
Duration: 400ms
Bezier Control Point: (0.5, -200)
Rotation: 15°
Trail Count: 3
Trail Interval: 50ms
```

---

### 2.3 连锁打出（Chain Play）

**触发时机：** 连锁满足条件，自动打出连锁卡牌

**动画描述：**
- **准备阶段（0–200ms）：** 连锁指示器亮起，卡牌依次从手牌滑出
- **触发阶段（200–600ms）：** 每张连锁卡牌沿预设轨道飞出，间隔 150ms
- **连锁结束：** 3张连锁卡在空中**汇聚成一束**，共同飞向目标
- **冲击特效：** 命中时屏幕左右两侧产生**元素色光柱**

---

## 三、连锁特效（Chain Effect）

### 3.1 连锁数积累显示

**动画描述：**
- 屏幕右上角出现连锁计数器
- 数字变化时：scale 1.0→1.4→1.0，ease-out
- 数字背景圆圈随连锁数增加逐步点亮（1→2→3→4→5）
- 4连锁+时计数器整体**发光脉冲**

---

### 3.2 连锁满级爆发（Full Chain）

**触发时机：** 达到5连锁满级

**动画描述：**
- **Phase 1（0–100ms）：** 屏幕中心出现白色十字闪光
- **Phase 2（100–400ms）：** 全屏元素色光波从中心向外扩散（radial gradient，scale 0→3）
- **Phase 3（200–600ms）：** 光环收缩，屏幕亮度短暂提升至 150%
- **Phase 4（400–800ms）：** 大量粒子从屏幕边缘飞入中心，卡牌从光柱中**缓缓升起**
- 背景音乐切换至高潮乐段（建议引用战斗高潮 BGM）

**参数：**
```
Full Chain Duration: 800ms
Screen Flash: 150% brightness, 100ms
Particle Count: 50
```

---

## 四、战斗反馈特效

### 4.1 伤害数字（Damage Number）

**动画描述：**
- 数字从目标位置**弹出**（scale 0→1.2→1.0，200ms）
- 根据伤害量显示不同颜色：
  - 普通伤害：白色
  - 暴击：红色 + "CRIT!" 文字
  - 治疗：绿色 + "+" 前缀
- 数字向上飘动并淡出（translateY -50px，opacity 1→0，duration 800ms）
- 暴击时数字有**震动效果**（random offset ±3px，3次，50ms间隔）

---

### 4.2 状态效果标识（Buff/Debuff）

**动画描述：**
- 效果施加时，目标头顶出现对应**图标+持续时间数字**
- 图标入场：scale 0→1.0，bounce easing，200ms
- 持续时间每秒递减，到最后3秒时数字**变为红色并闪烁**
- 效果消失时：图标向上飘出并淡出（opacity 1→0，400ms）

---

## 五、特效输出规格

### 文件格式
- **静态特效：** PNG Sequence（透明背景）
- **动态特效：** Lottie（JSON）/ Spine（.json + .png atlas）/ Unity Particle System

### 分辨率
- 消除特效：512×512（PNG序列，30fps）
- 打出特效：1024×1024（PNG序列，30fps）
- 全屏连锁特效：1920×1080（Lottie 或序列帧）

### 色彩模式
- 预乘Alpha（premultiplied alpha）
- sRGB 色域
