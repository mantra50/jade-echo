## 一、总体规范

### 1.1 时间单位
- 所有动画时长以 **秒（s）** 为单位
- 粒子特效持续时间含延迟销毁（通常 +0.3s）

### 1.2 缓动曲线速查表

| 曲线名称 | 公式（近似） | 适用场景 |
|---|---|---|
| ease-out | `1 - (1 - t)²` | 手牌入场、消除结束减速 |
| ease-in | `t²` | 手牌出牌、攻击飞射 |
| ease-in-out | `t < 0.5 ? 2t² : 1 - (-2t+2)²/2` | 血条变化、能量条变化 |
| spring | `sin(t * π * 4) * (1-t) + t` | 卡牌悬停弹跳、击中震动 |
| bounce | `abs(sin(t * π * 3)) * 0.3 + t * 0.7` | 伤害数字弹出 |
| linear | `t` | 持续性粒子（樱花飘落） |

## 二、消除特效（MatchVFX）

### 2.1 基础消除 — 3连匹配

```
时长：1.0s（含粒子消散 0.5s）
缓动：ease-out（减速停止）

视觉元素：
├── 主粒子群：樱花瓣（粉色 #FFB7C5 × 20-30片）
│   - 大小：随机 8-15px
│   - 旋转：每帧随机角速度
│   - 运动：向上飘散 + 横向随机漂移
│   - 生命周期：0.8s，opacity 从 1→0 线性衰减
│
├── 次粒子：墨迹消散（灰黑色 #2D2D3A × 8-12片）
│   - 大小：随机 12-20px，不规则形状
│   - 运动：向中心收缩 + 淡出
│   - 生命周期：0.5s
│
└── 中心光效：微小白光圈（直径 20px，渐显即逝）

预期效果：优雅、和风、有"消逝感"
```

### 2.2 4连消 — 剑气横贯

```
时长：1.2s
缓动：ease-in-out（先慢后快再减速）

视觉元素：
├── 主视觉：东方剑气（从起点到终点的光带）
│   - 颜色：渐变 #FFB7C5 → #FFFFFF → #FFD700（粉色→白→金）
│   - 形状：长条光带，宽 8-12px，长度 = 格子间距 × 4
│   - 拖尾：淡化的残影（3帧延迟拷贝）
│   - 中心亮斑：剑尖位置，白色高光
│
├── 粒子：金色能量碎片（#FFD700 × 15-20颗）
│   - 大小：3-6px
│   - 运动：沿剑气轨迹散射，带轻微重力下落
│   - 生命周期：0.6s
│
└── 打击音效配合（建议）：清脆剑鸣

预期效果：爽快感、力量感、视觉冲击强
```

### 2.3 5连消 / L型 / T型 — 终极消除

```
时长：1.5s
缓动：ease-in-out（强烈先慢后快）

视觉元素：
├── 主视觉：东方符阵爆发
│   - 中心：白色核心亮斑（扩张 → 消散）
│   - 外圈：紫粉旋转符文阵列（#9B5DE5 + #FFB7C5）
│   - 旋转：顺时针 360°，2圈
│   - 符文字形：简化东方古纹（"雷/火/风"抽象符文）
│
├── 冲击波：向外扩散的环形波纹（2道）
│   - 第1道：快（0.3s），白色半透明
│   - 第2道：慢（0.6s），紫粉半透明
│   - 透明度：从 0.6 → 0 线性衰减
│
├── 粒子：樱花 + 能量碎片（混合，量 × 1.5）
│   - 速度比基础消除快 30%
│   - 带轻微向心聚拢效果
│
└── 附加：屏幕轻微震动（0.1s，幅度 3px，ease-out）

预期效果：史诗感、高潮感、最强视觉反馈
```

## 三、伤害弹出（Damage Popup）

### 3.1 玩家受到伤害

```
时长：1.0s（弹出 → 悬停 → 消散）
缓动：bounce（弹出）+ ease-out（消散）

视觉元素：
├── 数字：红色（#E63946），粗黑体
│   - 字号：36px（普通伤害）/ 48px（高伤害 >10）
│   - 格式："-12"（前导负号）
│   - 动画：Y轴弹出距离 50px
│   - 弹出：前 0.2s bounce 曲线
│   - 悬停：0.2-0.7s 保持位置（轻微上下浮动 ±3px）
│   - 消散：0.7-1.0s opacity 1→0，Y轴继续上移 20px
│
└── 附加效果：受击闪烁（棋盘格闪红 0.1s）

预期效果：即时反馈、清晰可辨
```

### 3.2 敌人受到伤害

```
时长：0.9s
缓动：ease-out（全程减速）

视觉元素：
├── 数字：金色（#FFD700），粗黑体
│   - 字号：32px（普通）/ 44px（暴击 1.5x 大小 + "CRIT!"）
│   - 格式："-15"（前导负号），暴击时显示暴击值
│   - 动画：Y轴弹出距离 40px
│   - 弹出：ease-out 曲线（0.3s）
│   - 悬停：0.3-0.6s
│   - 消散：0.6-0.9s opacity 1→0
│
└── Boss 特殊：伤血数字 + 血条同步减少动画（0.5s ease-in-out）

预期效果：敌人受伤感知强、暴击有额外视觉奖励
```

### 3.3 治疗数字

```
时长：0.8s
缓动：ease-out

视觉元素：
├── 数字：绿色（#2A9D8F），粗黑体
│   - 字号：28px
│   - 格式："+20"（前导加号）
│   - 动画：Y轴弹出距离 35px
│   - 颜色脉冲：淡绿色 → 白色 → 淡绿色（0.4s 循环）
│
└── 特效：小型绿色粒子从数值位置向上飘散

预期效果：正向反馈、积极感
```

## 四、棋子移动动画（Gravity / Collapse）

### 4.1 棋子下落（重力塌陷）

```
时长：0.25s × 下落格数（每格 0.25s）
缓动：ease-in（加速下落，模拟重力）
最小时长：0.15s，最大时长：1.0s

视觉元素：
├── 移动轨迹：无残影（直接位置插值）
│   - 使用 Mathf.Lerp，步进 60fps
│
└── 落地：轻微弹跳（spring 曲线，幅度 3px，1次）

预期效果：流畅、响应快
```

### 4.2 棋子生成（从顶部填充）

```
时长：0.35s（每颗）
缓动：ease-out（从快到慢）

视觉元素：
├── 入场：从棋盘上方 50px 滑入到目标格
│   - 初始 opacity：0（渐显）
│   - 最终 opacity：1
│
└── 附加：淡入时轻微缩放（0.9 → 1.0，ease-out）

预期效果：自然、有填充感
```

### 4.3 棋子交换（Swap）

```
时长：0.2s（双向同时移动）
缓动：ease-in-out（平滑往返）

视觉元素：
├── 两个棋子同步互换位置
│   - 线性轨迹，无弧线
│   - 可加轻微缩放（0.95 → 1.0）表现"触碰感"
│
└── 失败交换（无效匹配）：弹回原位（0.15s spring）

预期效果：清晰、不拖沓
```

## 五、Boss 技能特效

### 5.1 BossGuardian — 冰冻格

```
时长：2.0s（施法 0.5s → 效果持续 1.5s）
缓动：ease-out（施法）→ linear（持续）

视觉元素：
├── 施法阶段：
│   - Boss 手持巨剑插向地面（0.5s ease-out）
│   - 冰晶裂纹从落点向外扩散（扇形，3道裂纹）
│   - 裂纹颜色：冰蓝色 #A8DADC → 白色 #FFFFFF
│
├── 效果阶段：
│   - 目标格子覆盖冰霜层（半透明冰蓝色方块）
│   - 冰霜层带有细小冰晶粒子（缓慢下落，模拟飘雪）
│   - 格子轻微发光（脉动，周期 1s）
│
└── 解除：冰霜裂纹碎裂消失（0.3s ease-in）

预期效果：压迫感、强占性、视觉清晰
```

### 5.2 BossMirror — 镜花水月

```
时长：2.5s（施法 0.8s → 幻象存在 1.0s → 消散 0.7s）
缓动：ease-in-out

视觉元素：
├── 施法阶段：
│   - 巨大镜像从 Boss 位置展开（扇形，弧度 120°）
│   - 镜面：半透明白银反光，流动水纹
│   - 符文：镜面边缘浮现旋转古纹
│
├── 幻象阶段：
│   - 复制体出现在玩家随机一列
│   - 复制体：半透明镜像（opacity 0.5），与真实棋盘重叠
│   - 幻象颜色偏蓝紫（#6B5B95 色调）
│
└── 消散：镜像碎裂成多边形碎片（8-12块），飞散消失

预期效果：迷惑性、东方神秘感
```

### 5.3 BossHourglass — 时间停滞

```
时长：3.0s（施法 1.0s → 停滞持续 1.5s → 恢复 0.5s）
缓动：ease-in-out

视觉元素：
├── 施法阶段：
│   - 巨大金色沙漏从天空落下（定位于棋盘上方）
│   - 沙漏边框：金色光芒，流动沙粒（#FFD700）
│   - 沙漏中央：棋盘所有棋子逐渐变成金色（0.8s ease-in-out）
│
├── 停滞阶段：
│   - 棋子全部变为金灰色，opacity 0.8
│   - 棋子周围出现时间冻结光环（金色细线圆圈）
│   - 沙漏持续倾倒，金色粒子缓慢下落
│   - 玩家 HUD 显示"时间停滞"提示
│
└── 恢复：金色褪去，棋子恢复彩色（0.5s ease-out）

预期效果：时间扭曲感、压迫感强
```

### 5.4 BossSpider — 蛛网标记

```
时长：1.5s（标记叠层 1.0s → 爆发 0.5s）
缓动：ease-out

视觉元素：
├── 标记：
│   - 被标记格子显示紫色蛛网图案（半透明叠加）
│   - 蛛网：8条放射线 + 同心圆（#9B5DE5）
│   - 每叠一层（Tick）蛛网发光强度增加（opacity 0.3 → 0.8）
│   - 格子边缘出现紫色小蜘蛛标记
│
├── 爆发：
│   - 蛛网从中心炸裂，紫光脉冲向外
│   - 所有被标记格子清除（等同于 3×3 消除特效）
│   - 紫色粒子向外飞散
│
└── 伤害数字弹出：紫色（#9B5DE5），格式标准

预期效果：危险性递增感、视觉上有积累感
```

## 六、卡牌效果特效

### 6.1 横斩（ClearRow）— 横向剑气

```
时长：0.8s
缓动：ease-in（加速向前）

视觉元素：
├── 光带：横向红色剑气（#E63946），沿目标行横贯
│   - 宽度：棋盘行高度
│   - 拖尾：3帧延迟残影（渐透明）
│
└── 命中：剑气到达末端时，白光爆发（1帧） + 棋子消失

预期效果：干脆、爽
```

### 6.2 星爆（ClearArea）— 3×3 爆发

```
时长：1.2s
缓动：ease-in-out

视觉元素：
├── 中心爆发：白光核心（快速扩张 → 消散）
│   - 扩张速度：前 0.4s 从 0 → 3×3 区域大小
│   - 颜色：白色核心 → #FFB7C5 边缘
│
├── 环形冲击波：2道（见终极消除规范）
│
└── 粒子：混合樱花 + 金色碎片（量 × 1.2）

预期效果：强烈、爆发感
```

### 6.3 能量灌注（EnergyBoost）

```
时长：1.0s
缓动：ease-out

视觉元素：
├── 起点：HUD 能量条位置
│   - 光点从能量条中心爆发（金色 #FFD700 小球 × 8-12个）
│
├── 运动：光点向外扩散后，飞向手牌区域
│   - 轨迹：弧线（向上弯曲），随机分散
│   - 球体大小：3-6px
│   - 球体颜色：金色，带轻微发光
│
└── 终点：手牌能量图标处，汇聚消失

预期效果：能量补充的满足感、正向反馈
```

### 6.4 元素爆弹（Bomb）

```
时长：1.5s
缓动：ease-in-out

视觉元素：
├── 准备：目标元素的所有棋子高亮（发光描边，0.3s）
│   - 颜色：目标元素对应色
│
├── 爆发：所有高亮棋子同时消失
│   - 消除特效（基础版）叠加出现
│   - 但无中心剑光，改为元素色能量圈
│
└── 收尾：棋盘剩余棋子自然下落填充

预期效果：精准、强烈、元素感
```
